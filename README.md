## Funkcje
* Menu
* Animacja menu
* Tablica o nieograniczonej wielkości z osiami
* Rysowanie wielokątów na tablicy
* Połączenie wierzchołków z przybocznym menu
* Tworzenie wielokątów z wierzchołków
